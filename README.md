# AI Chatbot using Tkinter & LangChain (Ollama)

## 📝 Overview

This is a simple AI chatbot built using **Tkinter** for the graphical user interface and **LangChain (Ollama)** for processing chatbot responses. The chatbot maintains conversation history and provides responses using the `deepseek-r1:1.5b` model.

## 🔧 Features

- Interactive chatbot with a **Tkinter GUI**
- **Scrollable chat display** with user & bot messages
- **Auto-scroll & read-only chatbox** for better readability
- **Formatted messages** (User: Blue, Bot: Green)
- **Maintains conversation history** for context-aware responses

## 🛠️ Installation


### 1️⃣ Clone the Repository

```sh
git clone https://github.com/RRaunakK/AI-Chatbot-Tkinter.git
cd AI-Chatbot-Tkinter
```

### 2️⃣ Install Dependencies

Ensure you have Python installed (3.8+ recommended). Then, install the required libraries:

```sh
pip install langchain langchain_ollama ollama 
```

### 3️⃣ Run the Chatbot

```sh
python chatbot.py
```

## 🖥️ Usage

1. Type your message in the input box.
2. Click **Send** or press `Enter` to submit.
3. The chatbot will respond based on conversation history.
4. Type **exit** to close the chatbot.
